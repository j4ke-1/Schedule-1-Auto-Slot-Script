# Schedule-1-Auto-Slot-Script

Controls & Script Functions
F4 → Stops the auto-spin loop
F5 → Completely exits the script

How It Works
The script rotates your camera to cycle between slot machines.
It automatically presses E to interact and start each slot.
Timing is optimized for continuous spinning and maximum efficiency.

Core Functions
MoveMouseRelative(dx, dy)
Moves your mouse cursor relative to its current position. Used to rotate the camera between slot machines.
pressE()
Simulates pressing the E key to activate slot machines.
